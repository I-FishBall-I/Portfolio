Dev Tools Domination

  • 主要的解題是要如何確認點選第一次~使用者按下shift所點選的範圍(主要的判斷)
  • //取出所有checkbox
const checkInput = Array.from(document.querySelectorAll('input'))                                    
let lastChecked = null;
    //按下shift的判斷
    function selectMany(e) {
        /*
        第一層判斷
            當第一次按下先記錄當前索引，否則清空
        第二層
            按下shift且第一次不為空則記錄第二次索引
            在對checkInput陣列進行slice(比較索引小的放前面)
            對被切出來的陣列跑foreach把每個input勾選起來
        */
        if (this.checked) {
            if (e.shiftKey && lastChecked !== null) {
                let nowCheck = checkInput.indexOf(this)
                checkInput.slice(
                    Math.min(nowCheck, lastChecked), 
                    Math.max(nowCheck, lastChecked))
                .forEach(input => input.checked = true);
            }
            lastChecked = checkInput.indexOf(this);
        } else {
            lastChecked = null;
        }
}
    //對每個checkbox加上監聽
    checkInput.forEach(input => input.addEventListener('click', selectMany));

Slide in on Scroll

  • 監聽scroll讓圖片滑到指定位置決定他該出現還是消失，主要用到window.scroll、offsetTop及clientHeight
  • window.addEventListener('scroll', checkImg)
function checkImg() {
    const
        windowTop = window.scrollY,  //畫面最頂部的位置
        windowBottom = window.scrollY + innerHeight;//取得畫面最底部的位置
    document.querySelectorAll('img').forEach(item => {
        const
            imgTop = item.offsetTop + item.clientHeight / 4,//圖片1/4處
            imgBottom = item.offsetTop + item.clientHeight * 3 / 4,//圖片3/4處
            imgEnterScreen = windowBottom > imgTop,//如果底部位置超過圖片1/4處則為TRUE讓圖片進入
            imgInScreen = windowTop < imgBottom;//如果頂部位置超過圖片3/4處則為False讓圖片出去
        imgEnterScreen && imgInScreen ? item.classList.add('active') : item.classList.remove('active');
    })
}
